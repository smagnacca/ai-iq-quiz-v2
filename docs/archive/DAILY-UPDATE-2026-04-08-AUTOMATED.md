## Key Lessons Added to Best Practices Log

### Lesson 24 — Google Sheets Column Alignment
Column schema between `submit-lead.js` (writer) and `check-followups.js` (reader) must match exactly. A 4-column vs 7-column mismatch caused duplicate email batches. Maintain single source of truth for column schema.

### Lesson 25 — Loop + Break Guard in Schedulers
Never use flat `if/if/if` in a cron job that processes ordered steps — all overdue steps fire at once. Use `for...of` + `break` to send exactly one step per run.

### Lesson 26 — Sandbox Can't Push to GitHub
The Cowork sandbox blocks outbound GitHub traffic (HTTP 403 proxy). Git push from sandbox will always fail. Use computer-use to trigger the scheduled deploy task instead, or verify current deploy state via Netlify MCP.
